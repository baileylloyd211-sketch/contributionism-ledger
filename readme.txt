# Contributionism Ledger

Early prototype of a transparent contribution ledger inspired by Contributionism — a framework to recognize and incentivize high-impact work in critical infrastructure without displacing markets.

## Purpose
This repo explores a simple, verifiable ledger for tracking contributions (e.g., in utilities like Portland Water Bureau or BES). Future versions could include:
- UIAC-style NOO calculations on public data
- Mock CS (Contribution Score) calculator
- Dashboard for resilience investment flows

## Current State
- Vite + React frontend
- Basic ledger component (Ledger.jsx)
- Deployed demo: https://contributionism-ledger.vercel.app (WIP)
